# METRO DASH

A bright, cartoony **endless runner** with a full game meta — coins, high score, shop, characters, boards, power-ups, missions, and a daily reward. One self-contained `index.html`, no build, no dependencies. Original art in the endless-runner format (not affiliated with any existing game).

## Play
- **← →** switch lanes · **↑ / Space** jump · **↓** roll
- **Mobile** — swipe left/right/up/down (tap = jump)

Dodge **trains** (switch lane), hop **barriers** (jump), roll under **gates**, grab **coins**, snag **power-ups**: 🧲 magnet, 🚀 jetpack, ✖️ 2× coins, 👟 super-sneakers.

## Meta
- Coin economy (banked after each run), high score + top-run
- **Shop**: unlock characters + boards, upgrade power-up durations
- **Missions** (rotating) and a **daily reward** with streaks
- Everything persists in localStorage

## Tech
Canvas 2D pseudo-3D lane projection, WebAudio SFX (no files), PWA (network-first SW).
